    Códigos HTML

    Html: é uma linguagem que estrutura as páginas da web e as deixa editaveis

    Tags: usadas como funções do html

    • html: Define o início e o fim do documento HTML.
    • head: Contém informações sobre o documento HTML, como título, metadados e links para estilos CSS.
    • p: Define um parágrafo de texto.
    • a: Define um hiperlink para outra página ou recurso. O atributo "href" especifica o URL de destino.
    • img: Insere uma imagem na página. O atributo "src" especifica o caminho para a imagem.
    • div: Define uma divisão ou seção genérica em um documento HTML. É frequentemente usado como um contêiner para outros elementos HTML
